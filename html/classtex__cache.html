virtual

Access function for tex_cache return values: RESERVATION_FAIL if request could not be accepted otherwise returns HIT_RESERVED or MISS; NOTE: never returns HIT since unlike a normal CPU cache, a "HIT" in texture cache does not mean the data is ready (still need to get through fragment fifo)
